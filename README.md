# Магазин фирменной и сувенирной продукции

Магазин фирменной и сувенирной продукции с веб-интерфейсом и бэкендом на Python.

## Структура проекта

- `backend/` — бэкенд на Python (FastAPI/Flask)
  - `main.py` — основной файл приложения
  - `db.py` — работа с базой данных
- `index.html` — главная страница
- `scripts/` — клиентские JavaScript-скрипты
- `style/` — CSS-стили
- `img/` — изображения

## Установка и запуск

### Бэкенд

```bash
cd backend
pip install -r requirements.txt
python main.py
```

### Фронтенд

Откройте `index.html` в браузере.
